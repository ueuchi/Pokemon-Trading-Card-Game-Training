## 1. 用語定義

| 用語 | 説明 |
|------|------|
| バトル場（バトルゾーン） | 自分のアクティブポケモンを置くゾーン。1枚のみ |
| ベンチ（ベンチゾーン） | 控えのポケモンを置くゾーン。最大5枚 |
| 山札 | プレイヤーが使用するカードのデッキ |
| 手札 | 山札から引いた、プレイヤーが手に持つカード |
| サイドカード | 対戦開始前に山札から引いて裏向きで置く6枚のカード |
| トラッシュ | 使用済み・きぜつしたカードの捨て札置き場 |
| たねポケモン | 進化していない基本のポケモンカード |
| 1進化ポケモン | たねポケモンから進化する段階のポケモン |
| 2進化ポケモン | 1進化ポケモンからさらに進化する段階のポケモン |
| ダメージカウンター | ポケモンに与えられたダメージを示すカウンター（10ダメージにつき1個） |
| きぜつ | ポケモンのHPが0になった状態。そのポケモンをトラッシュする |

## 2. デッキ構成ルール

- デッキは **60枚** で構成する
- 同名カードは **4枚まで** 入れることができる（基本エネルギーを除く）
- デッキには **たねポケモンを最低1枚** 入れなければならない

## 3. 対戦前の準備ルール

以下の手順を **上から順番に** 実行する。

### 手順1: 先行・後攻の決定

- **コイントス** でランダムに先行・後攻を決定する
- 先行プレイヤーは最初のターンに攻撃宣言ができない

### 手順2: 初期手札を引き、ポケモンを場に出す

- 両プレイヤーは山札から **7枚** カードを引く
- 手札の中からたねポケモンを選び、**バトルゾーンに1枚**、**ベンチゾーンに好きな枚数（0〜5枚）** を **裏向き** で場に出す

### 手順3: マリガン処理

- 手札にたねポケモンが **1枚もない場合**、相手プレイヤーにその手札を見せ、山札にカードをすべて戻してシャッフルし、再度7枚引く（**マリガン**）
- 両プレイヤーともたねポケモンが引けるまでこれを繰り返す
- マリガンが発生した場合、**相手プレイヤーはマリガンが発生した回数分、追加でカードを引くことができる**（引くか引かないかを選択できる）
- マリガン分の追加カードを引くタイミングは、**サイドカードを置いた後**とする

### 手順4: サイドカードを置く

- 両プレイヤーは山札からカードを **6枚** 引き、**サイドカードゾーンに裏向き** で置く

### 手順5: マリガン追加ドロー

- 手順3で発生したマリガン分の追加カードを、この時点で引く（引くかどうかを選択）

### 手順6: ゲームスタート

- スタートの合図で、バトル場とベンチに出ているポケモンカードを **表向き** にする
- 先行プレイヤーのターンから開始する

## 4. 対戦時のルール

### 4-1. ターン開始時のドロー

- ターン開始時、プレイヤーは **山札からカードを1枚引く**
- ターン開始時に山札が0枚で引けない場合、そのプレイヤーの**負け**となる

### 4-2. ポケモンを場に出す

- バトル場には **1枚のポケモン** のみ置くことができる
- ベンチには **最大5枚** のポケモンを置くことができる
- 場に出すことができるのは **たねポケモンのみ**（特殊な特性を持つポケモンを除く）

### 4-3. 進化ルール

- 進化の順序は **たねポケモン → 1進化ポケモン → 2進化ポケモン** の順に、カードを重ねることで進化する
- 段階を飛ばした進化はできない（たねポケモンから2進化へは直接進化不可）
- **以下の場合は進化できない**:
  - 最初のターン（先行・後攻ともに）
  - その試合中に場に出した **同一ターン内**（次のターン以降から進化可能）
- 進化すると、以前のポケモンについていた特殊状態はリセットされる
- 進化しても付与されているエネルギーはそのまま引き継がれる

### 4-4. エネルギーの付与

- エネルギーカードは **1ターンに1回**、手札から場に出ているポケモン（バトル場・ベンチ問わず）に付与することができる

### 4-5. トレーナーカードの使用

| カテゴリ | 使用制限 | 説明 |
|---------|---------|------|
| サポート | 1ターンに1枚まで | 強力な効果を持つ。使用後はトラッシュ |
| グッズ | 無制限 | 何枚でも使用可能。使用後はトラッシュ |
| スタジアム | 1ターンに1枚まで | 場に1枚のみ置ける（後述） |

### 4-6. スタジアムカードの使用

- スタジアムカードは **1ターンに1枚まで** 使用できる
- スタジアムカードを場に出す際、すでに別のスタジアムカードが置かれていた場合は、**既存のスタジアムカードをトラッシュ** してから新しいスタジアムを置く

### 4-7. ポケモンを逃がす

- ポケモンを逃がすことができるのは **1ターンに1度のみ**
- 逃げるには、そのポケモンのカードに記載された **逃げエネルギーのコスト分のエネルギーをトラッシュ** する必要がある
- 逃げた後は、ベンチのポケモンをバトル場に出す

### 4-8. 攻撃ルール

#### 攻撃宣言の条件

- 攻撃宣言は **バトル場に出ているポケモンのみ** 行うことができる
- 攻撃に必要なエネルギーがそのポケモンに付与されている場合に **攻撃宣言** ができる
- **先行プレイヤーは最初のターンに攻撃宣言ができない**

#### 攻撃後の処理

- 攻撃を行うと **そのターンは終了** し、相手のターンへ移行する

#### ダメージ計算

攻撃には以下の2種類がある。

**① 通常ダメージ**

弱点・抵抗力を計算してダメージを決定する。

```
最終ダメージ = 攻撃のダメージ × 弱点倍率 - 抵抗力
```

| 条件 | 計算 |
|------|------|
| 相手ポケモンの弱点タイプと攻撃タイプが一致 | ダメージ × 2 |
| 相手ポケモンの抵抗力タイプと攻撃タイプが一致 | ダメージ - 30 |
| 両方該当する場合 | (ダメージ × 2) - 30 |

- 攻撃するポケモンの **タイプ** が攻撃のタイプとなる

**② ダメージカウンター型**

- 弱点・抵抗力の計算は **行わない**
- 指定された数のダメージカウンターを相手のポケモンに乗せる
- ダメージカウンター1個 = 10ダメージ相当

### 4-9. バトル場のポケモンがきぜつした後の処理

- バトル場のポケモンが相手に倒されトラッシュした場合、**倒されたプレイヤー** が自らベンチからバトル場に出すポケモンを選ぶ
- ベンチにポケモンがいない場合、そのプレイヤーの**負け**となる（バトル場が空）

### 4-10. きぜつとサイドカードの取得

- ポケモンのHPが **0以下** になるとそのポケモンは **きぜつ** する
- きぜつしたポケモンは、付与されているエネルギーカードとともに**トラッシュ**する
- きぜつさせたプレイヤーは、きぜつしたポケモンのカードに記載されたルールに従い **自分のサイドカードを引く**
  - 通常ポケモン: サイドカードを **1枚** 取得
  - ルールを持つポケモン（例: ex、V など）: **複数枚** 取得（カードに記載）

## 5. 特殊状態（仕様のみ）

> ⚠️ **実装対象外**: 特殊状態は **Phase 9以降** で実装する。本ドキュメントでは仕様のみを記載する。

以下の特殊状態が存在する。

| 特殊状態 | 概要 |
|---------|------|
| どく | 毎ターン終了時にダメージカウンターが乗る |
| やけど | 毎ターン終了時にダメージカウンターが乗る。コイントスで回復判定 |
| こんらん | 攻撃時にコイントスが発生し、失敗すると自分にダメージ |
| まひ | 次のターン、攻撃も逃げることもできない |
| ねむり | 次のターン、攻撃も逃げることもできない。コイントスで回復判定 |
| 逃げられない | そのターン中、ポケモンが逃げることができない |

- 特殊状態は **1つのポケモンに1つ** しか付与できない（後からかかった状態が上書きされる）
- ポケモンが進化・退化すると特殊状態はリセットされる

## 6. 勝利条件

以下のいずれかの条件を満たしたプレイヤーが **勝利** となる。

| 条件 | 説明 |
|------|------|
| サイド全取得 | 自分のサイドカード6枚をすべて取得する |
| 相手のバトル場が空 | 相手のバトル場にポケモンがいない状態になる（ベンチも空の場合） |
| 山札切れ | ターン開始時に相手が山札からカードを引けない状態になる |
